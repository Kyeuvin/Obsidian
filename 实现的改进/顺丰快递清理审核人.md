#### A:
在230后台`E:\SVN\B2B平台\B2BSite.Manage\tool\UpdateExpressFile.aspx`里加了个清理审核人按钮，实现SQL语句功能。
实现的SQL语句：
```
UPDATE BillSaleProduct SET CHECKEDBY = NULL WHERE productrelation3 = '顺风速运' AND ProductName = '旅游-其它' AND CHECKEDBY = '操作人' AND creattime > @startDate AND creattime < @endDate

//start和end的日期时前面日期选择框的日期
```
